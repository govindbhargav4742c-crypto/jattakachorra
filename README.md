# 📊 SalesFlow Dashboard

A modern, interactive sales dashboard built with **HTML5**, **CSS3**, and **JavaScript** featuring real-time data visualization with Chart.js.

## 🎯 Features

- **📈 Dashboard Page** - Overview of total sales, average orders, and revenue trends
- **💰 Revenue Analytics** - Weekly and monthly revenue tracking with visual charts
- **👥 Customer Analytics** - Track total customers, new signups, and growth metrics
- **⚙️ Settings** - Configure dashboard preferences and account settings
- **📱 Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **🎨 Modern UI** - Dark sidebar with clean, professional layout
- **📊 Interactive Charts** - Beautiful gradient line charts powered by Chart.js
- **⚡ Fast Performance** - Lightweight and optimized for speed
